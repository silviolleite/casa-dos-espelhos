# casa-dos-espelhos
Trabalho de Estrutura de dados. 

O exercício “Casa dos Espelhos” foi um desafio proposto para auxiliar o desenvolvimento teórico e prático dos alunos na disciplina de Estrutura de Dados e consiste em desenvolver um software que solucione o problema e determine o número de possibilidades dados os números de espelhos e os números de reflexões, as condições são que as reflexões podem acontecer na descida ou na subida dos raios e podem ficar refletindo entre dois espelhos.

A linha de estudo em princípio era a recursividade tendo em vista que a sequência de Fibonacci respondia as possibilidades com três espelhos, mas no decorrer dos estudos solucionamos o problema com estudo de matrizes. O foco foi encontrar uma única formula que solucionasse qualquer possível números de espelhos e de reflexões.


A linguagem escolhida para o desenvolvimento do software foi o C++, isso porque a resolução deste problema requer uso muito alto de processamento e o C++ pode ser usada para programar qualquer tipo de hardware, desde os mais simples até os mais complexos. Além disso, C++ é uma linguagem que gera programas em código de máquina, que funcionam com ou sem a participação de sistemas operacionais no dispositivo.
